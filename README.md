# 🌟 Aurelia

> Your AI-powered personal finance advisor.

Aurelia lets you upload your financial documents and chat with them. Ask questions, track spending, and get smart predictions — all in one place.

---

## Tech Stack

- **Frontend:** React
- **Backend:** Jakarta EE
- **AI Service:** Python · FastAPI · LangChain
- **Database:** PostgreSQL + ChromaDB

## Features

- Upload bank statements & invoices
- Chat with your finances using RAG
- Automatic transaction categorization (ML)
- Spending forecasts & anomaly detection
